# Side Panel

[Chrome Docs](https://developer.chrome.com/docs/extensions/reference/sidePanel/)

:::tip Chromium Only
Firefox does not support sidepanel pages.
:::

<EntrypointPatterns
  :patterns="[

## Definition

Plain old HTML file.

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Title</title>
  </head>
  <body>
    <!-- ... -->
  </body>
</html>
```
